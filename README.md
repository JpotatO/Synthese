Synthese
========

Synthese is an open source application for Windows written in vb.net that gives you the oppertunity to bring web apps to your desktop.

Features
--------
*Trident Engine (Gecko is coming!)
*Custom Icon
*Window Style
*Choose shortcut location (win7 only)
